Download
    You can download the latest version of Vegastrike from here, and find links to some of the less comon libraries in use. If you want to find earlier versions of Vegastrike, visit our SourceForge Project page .

    The current version of Vegastrike is 0.2.1

Windows Files

  • Vegastrike 0.2.1 Windows Beta (34.2 M)

    Or you can download the smaller version, without the terrain files present.

  • Vegastrike 0.2.1 Windows Slim Beta (No terrain) (25 M)

Linux Files
    You can grab either the source code:

  • Vegastrike 0.2.1 Source Code (548 KB)

    • and build the binary, or take the prebuilt binary (note, the prebuilt does not contain all the features found in the 0.2.1 source):

  • Vegastrike 0.2.0 Linux Binary (1.6 M needs glibc2.2)

    • Place the binary in the data directory:

  • Vegastrike 0.2.1 Unix Data (30.6 M)

    • and then run! Alternatively, you can use the Suse RPMS (also a slightly older version):

  • Vegastrike 0.2.0 RPMs for SuSE 7.3 (size about 30M) *thanks* to Stefan Dirsch
