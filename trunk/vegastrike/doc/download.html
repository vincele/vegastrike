Vegastrike New Version
  • All future releases will be made via the sourceforge website!
  • Windows people need only the data.zip file and can run the .exe file from there.
  • Linux users need to be sure they run the compiled source code (vegastrike) from the data.zip directory, so that the program can find light.bmp and the host of other files that should be contained there. If vegastrike is run from an improper directory it will print "light not found" and then segfault until some future release where I print a nice friendly error message..then quit.
