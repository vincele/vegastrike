The Mission Selector is a nice tool to start Vegastrike from a GUI.

Start it by going to the data dir and type: ./vsselect


 

The Mission Selector will show you a list of all available missions.

In this screenshot, the mission 'proving_grounds' has been selected.

On the right side of the list, information about the mission is displayed.

Click 'Run Mission' to start the mission.
