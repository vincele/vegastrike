Force Feedback support for Vegastrike is currently being developed for Linux.

It is my task to write/help write a high-level force-feedback library, based on the kernel low-level drivers by Johann Deneux.

Force Feedback for Linux only works with I-Force2.0 compliant devices, see the Linux and I-Force page.

Current status of development:

  • force feedback works for me on linux
    using kernel 2.4.17, Johann's latest patch, the joystick connected via RS232
    Johann's testing programs work


Many *thanks* to AVB for providing open source software developers with free samples of their force-feedback devices, to support them in writing drivers/libraries etc.

Especially thanks to June Yu from AVB and to Johann Deneux.
