Installing Vegastrike on Linux:

1. go to the download page and download from there the latest unixdata tar file and the latest linux binary

2. extract the binary to a directory of your choice
    example: tar zxf ../vegastrike-0.1.3-x86-linux.tar.gz

3. cd to the created subdirectory
    cd vegastrike-0.1.3

4. extract the content from the unixdata tarball into this directory:
    tar zxf ../vegastrike_unixdata-0.1.3.tgz

5. then cd to the data directory

6. to start vegastrike, type
    ./vegastrike.sh <mission-file>

    Examples:  ./vegastrike.sh     (starts the default mission)
                      ./vegastrike.sh mission/ships_confed1.mission
                      ./vegastrike.sh mission/test/test4.mission

Please send me Email if you're having any problems.
 

Installing Vegastrike on Win32:

1. download the windows binary zip file

2. unzip the file in a directory of your choice
